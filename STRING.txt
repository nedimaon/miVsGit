
append(chars)
Добавляет заданную строку (в символах) к строке
 Например, 'Hello' append 'World' превращается в 'HelloWorld'

between(left, right)
Извлекает строку между левой и правой строками
 Например, 'The quick brown fox
' между 'quick' и 'fox' превращается в ' brown'

camelize()
Удаляет все подчеркивания и тире и преобразует строку в верблюжий регистр
 Например, 'data_rate' превращается в 'dataRate'

capitalize()
Делает первый символ строки заглавным
 Например, 'jon' превращается в 'Jon'

chompLeft(prefix)
Удаляет префикс из начала строки
 Например, 'foobar' chompLeft 'foo' превращается в 'bar'

chompRight(suffix)
Удаляет суффикс из конца строки
 Например, 'foobar' chompRight 'bar' превращается в 'foo'

collassWhitespace()
Преобразует все смежные пробелы в один пробел

contains(find)
Возвращает true, если строка содержит find
 Например, 'JavaScript — один из лучших языков
' contains 'one' — true

count(find)
Возвращает количество вхождений find
 Например, 'JP любит программировать
 JP не играет в НБА
' count 'JP' — это число 2

dasherize()
Возвращает преобразованную строку в «верблюжьем» регистре в строку, разделенную дефисами
 Например, 'dataRate' превращается в 'data-rate'

decodeHTMLEntities()
Декодирует HTML-сущности в их строковое представление
 Например, «Ken Thompson & Dennis Ritchie» преобразуется в «Ken Thompson & Dennis Ritchie»

decodeURIComponent()
Декодирует компонент URI
 Например, «http%3A%2F» преобразуется в «http://»

delLeftMost( find )
Возвращает оставшуюся строку после удаления первого вхождения find и всего, что находится слева от find при сканировании <b>слева направо</b>
 Если find не найден, возвращается неизменённая строка
 Например, «\Parent\child\folders» delLeftMost '\child' оставляет '\folders'

delRightMost( find )
Возвращает оставшуюся строку после удаления первого вхождения find и всего, что находится справа от find при сканировании <b>справа налево</b>
 Если find не найден, возвращается неизменённая строка
 Например, '\Parent\child\folders' delRightMost '\child' оставляет '\Parent'

encodeURIComponent()
Кодирует компонент URI
 Например, 'http://' становится 'http%3A%2F'

endsWith(chars)
Возвращает true, если строка заканчивается символами 
 Например, 'hello jon' endsWith 'jon' — true

escapeHTML()
Экранирует HTML
 Например, '&lt;div&gt;hi&lt;/div&gt;' становится '&lt;div&gt;hi&lt;/div&gt;'

ruberLeft(prefix)
Гарантирует, что строка начинается с префикса 
 Например, 'subdir' ensureLeft '/' становится '/subdir'

EnsureRight(suffix)
Гарантирует, что строка заканчивается суффиксом
 Например, 'dir' ensureRight '/' преобразуется в 'dir/'

getRightMost(find)
Находит первое вхождение find и возвращает все элементы справа от find (исключая сам find) при сканировании <b>справа налево</b>
 Если find не найден, возвращается неизмененная строка
 Например, '\Parent\child\folders' getRightMost '\child' — это '\folders'

getLeftMost(find)
Находит первое вхождение find и возвращает все элементы слева от find (исключая сам find) при сканировании <b>слева направо</b>
 Если find не найден, возвращается неизмененная строка
 Например, '\Parent\child\folders' getLeftMost '\child' — это '\Parent'

humanize()
Преобразует входные данные в понятный для человека формат
 Например, «capitalize dash-CamelCase_underscore trim» становится «Capitalize dash camel case underscore trim»

isAlpha()
Возвращает true, если строка содержит только буквы

isAlphaNumeric()
Возвращает true, если строка содержит только буквы и цифры

isEmpty()
Возвращает true, если строка состоит только из пробелов или имеет значение null/undefined

isLower()
Возвращает true, если символ или строка в нижнем регистре

isNumeric()
Возвращает true, если строка содержит только цифры

isUpper()
Возвращает true, если символ или строка в верхнем регистре

latinise()
Удаляет диакритические знаки с латинских символов
 Например, «crème brûlée» становится «creme brulee»

left( n )
Возвращает подстроку, обозначенную n положительными крайними слева символами
 Например, 'My name is JP', left 2 is 'My'
 
length()
Свойство возвращает длину строкового объекта
 Например, длина 'hi' равна 2
 
lines()
Возвращает массив строк
 Совместимо с различными платформами

pad( length , char )
Дополняет строку по центру указанным символом char
 Символ char может быть строкой
 Например, 'hello' pad '
' — это 'hello'

padLeft( length , char )
Дополняет строку слева
 Например, 'hello' padLeft '
' — это 'hello'

padRight( length , char )
Дополняет строку справа
 Например, 'hello' padRight '
' — это 'hello'

parseCSV()
Преобразует строку CSV в массив
 Например, "'a','b','c'" преобразуется в ['a', 'b', 'c']

repeat(n)
Возвращает строку, повторенную n раз
 Например, '*' repeat 5 возвращает '*****'

replaceAll(search, replace)
Возвращает новый
Строка, в которой все вхождения search заменены на replace
 Например, 'does IT work?' replaceAll ' ' '_' становится '_does_IT_work?_'

right( n )
Возвращает подстроку, обозначенную n положительными крайними правыми символами
 Например, 'My name is JP' right 2 is 'JP'

setValue( chars )
Устанавливает содержимое равным chars 

slugify()
Преобразует текст в допустимый URL-слаг
 Удаляет акценты с латинских символов
 Например, 'The Quick Brown Fox' становится 'the-quick-brown-fox'

split( separate , limit )
Разделяет строку на массив подстрок, используя строку-разделитель для определения места каждого разделения
 Ограничение на количество разделителей можно указать в необязательном параметре limit 
 Например, 'Hello planet earth' split ' ' возвращает ['Hello', 'planet', 'earth']

splitLeft( separater , maxSplit , limit )
Возвращает массив строк, разделенных слева по разделителю
 Выполняет не более maxSplit разделений и преобразует результат в массив с не более чем limit элементами
 Например, 'We built this city' splitLeft ' ' и 1 преобразуется в ['We', 'built this city']

splitRight( separater , maxSplit , limit )
Возвращает массив строк, разделенных слева по разделителю
 Выполняет не более maxSplit разделений и преобразует результат в массив с не более чем limit элементами
 Например, 'We built this city' splitLeft ' ' и 1 преобразуется в ['We built this', 'city']

startingWith( prefix )
Возвращает true, если строка начинается с prefix 
 Например, «Здесь был Стив» начинается с «Стив» — true

strip(chars)
Удаляет символы из заданной строки
 Например, «Hello World» strip превращается в «HelloWorld»

stripPunctuation()
Удаляет все знаки препинания
 Например, «My, st[ring] *full* of %punctuation)» превращается в «My string full of punctuation»

stripTags(tag)
Удаляет весь HTML-код с указанным тегом или все теги, если тег пуст
 Например, «&lt;p&gt;just &lt;b&gt;some&lt;/b&gt; text&lt;/p&gt;» stripTags превращается в «just some text»

template(values, open, close)
Принимает строку и интерполирует значения
 По умолчанию открыто на {{ и закрыто на }}, если open и close оставлены пустыми для шаблонов, совместимых с Mustache
 Однако вы можете изменить это значение по умолчанию, указав open и close
 Например, шаблон 'Hello {{planet}}!!!' {"planet":"Earth"} становится 'Hello Earth!!!'

titleCase()
Изменяет строку, делая первую букву каждого слова заглавной, включая слова с дефисом
 Например, 'hello world' titleCase становится 'Hello World'

toBoolean()
Преобразует строку логической истины в логическое значение
 Строка, содержащая 'true', 'on', 'yes' или '1', станет логическим значением true, в противном случае — false

toCSV( JSON )
Преобразует массив или объект в строку CSV

toFloat( precision )
Анализирует строку и возвращает число с плавающей точкой с указанной точностью
 Например, '3
45522' toFloat 2 преобразуется в 3
45

toInteger()
Разбирает строковый аргумент и возвращает целое число
 Например, '5
3' toInteger преобразуется в 5

toLowerCase()
Меняет регистр всей строки на нижний
 Например, 'Hello world' toLowerCase преобразуется в 'hello world'

toString()
Преобразует текущее значение в строку
 Например, 123 toString преобразуется в '123'

toUpperCase()
Меняет регистр всей строки на верхний
 Например, 'Hello world' toUpperCase преобразуется в 'HELLO WORLD'

trim()
Удаляет начальные и конечные пробелы из строки
 Например, ' hello ' trim преобразуется в 'hello'

trimLeft()
Удаляет начальные пробелы из строки
 Например, «Привет там»
 trimLeft становится «Привет там»

trimRight()
Удаляет конечные пробелы из строки
 Например, «Привет там»
 trimRight становится «Привет там»

truncate(length, chars)
Усекает строку с учётом расположения слов и количества символов
 Например, «Это длинный текст»
 truncate 14, и «Подробнее» становится «Подробнее»

underscore()
Преобразует строку в «верблюжьем» регистре в строку, разделённую подчёркиваниями
 Например, «dataRate» с подчёркиванием становится «data-rate»

unescapeHTML()
Деэкранирует HTML
 Например, «&lt;div&gt;hi&lt;/div&gt;» unescapeHTML преобразуется в '&lt;div&gt;hi&lt;/div&gt;'
wrapHTML( tag , attributes )
Строка будет заключена в тег HTML-элемента, а его атрибуты будут представлены в формате JSON
 Например, 'Venkat' wrapHTML 'div' и { "class": "left bullet" } преобразуются в '&lt;div id="content" class="left bullet"&gt;Venkat&lt;/div&gt;'
